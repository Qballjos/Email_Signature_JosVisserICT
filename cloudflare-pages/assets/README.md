## Specifications

- **Format:** PNG with transparent background ✅
- **Brand color:** Orange (#ea8b39) ✅
- **Usage:** Email signatures, websites, documents, social media ✅
- **Compatibility:** All modern browsers and email clients ✅

## To Update Logos

1. Replace the appropriate PNG file in this folder
2. Push to GitHub
3. Cloudflare will auto-deploy (1-2 minutes)

## Notes

- App Icon is used in email signature (90×90px display)
- Artboard logo is used in legal page headers (60×60px display)
- Favicon is used for browser tabs
- All logos maintain brand consistency across platforms
