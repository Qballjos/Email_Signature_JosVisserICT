# Logo Assets

## Required File

Place your logo file here:

- **logo.png** - Your logo that works for both light and dark themes

## Specifications

- **Format:** PNG with transparent background
- **Dimensions:** 120x120 pixels (square)
- **File size:** Under 100KB (smaller is better)
- **Design:** Should be visible on both light and dark backgrounds

## Current Files

Your logo is suitable for both light and dark themes, so only one file is needed.

### Upload Location

This file should be deployed to:
```
https://static.josvisserict.nl/assets/logo.png
```

## Tips

- Use colors that work on both light and dark backgrounds
- Test visibility on white and dark backgrounds
- Keep file size small (current: ~5KB is excellent!)
- PNG format ensures maximum compatibility with email clients
